5 o'Clock Somewhere

My first project working in a team with two other contributors. Our goal was to develop a simple application that could interact with at least two public HTTP APIs. In addition, we attempted to seamlessly integrate as many different technologies we learned over the first 8 weeks of our course. When you enter the site, we greet the user with a Modal, request to obtain the user's location via the Geolocation API and pass the location coordinates to the Yelp API call to display local cocktail bars to the user. The user can proceed to look up any ingredient or cocktail name via the search bar and the page will dynamically display all the drinks and recipes that meet the user's search. It will also save recently searched items into a dynamically generated nav bar via local storage.

Project Technologies

  • Group combined effort
  • Calls two HTTP APIs
  • Local Storage
  • jquery / Javascript
  • Materialize CSS Framework
  • Local Storage, Geolocation APIs
  • HTML & CSS

Weather App

My weather app project is very special to me. It was my first creative example of utilizing everything I learned in the first 6 weeks of our coding course. On page load, it requests the user's location via the Geolocation API and passes those coordinates to the Public Weather API which then dynamically creates elementes on the page displaying the user's local weather. If a user searches for a city, the page makes the same API call and dynamically manipulates the elements. It also stores recently searched cities into the nav bar and locally on the user's browser.

Project Technologies

  • jQuery / Javascript
  • AJAX HTTP Call to Public Weather API
  • Geolocation & Local Storage APIs
  • Bootstrap CSS Framework
  • Html & CSS

Day Planner App

This day planner application was my second successful attempt at utilizing dynamic Javascript elements. The page will tell you the present date and change the background color of each hour of the work day from green (future state) to red (present hour) to grey (past hour). User's can type into the textareas and click the blue save buttons on the right to save that data to local storage in their browser. Items will remain there until the user deletes browser history.

Project Technologies

  • jQuery / Javascript
  • Local Storage API
  • Moment.JS Library
  • HTML & CSS
